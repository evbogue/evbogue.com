<p><strong>This is how I write on my computer:</strong></p>
<p>I use <a href="http://www.literatureandlatte.com/scrivener.php">Scrivener</a> to write everything now. ($45, with a free 30-day trial)</p>
<p>The team at Literature and Latte have been incredibly on the ball with updates to this App. </p>
<p>Using Mac OS Lion, I put Scrivener in full screen mode. In this way, I don’t get distracted by the rest of the things going on in my computer. </p>
<p>My computer is an entrance to the dream world of the Internet. Scrivener is my own private world which isn’t quite in the dream. It’s going deep, but just with my internal self and the canvas of the typed page.</p>
<p>I organize all of my writing into folders in a Scrivener Document called “Notebook”. I have an icon on my desktop (it’s one of only a few icons on my desktop) which leads directly into Notebook. These folders change. </p>
<p>When I want to write something for the blog, I create a blank document in my ‘Blog Drafts’ folder. </p>
<p>When I want to write something for <a href="/posts/letter">Evolving Your Digital Work</a>, I create a blank document in my ‘Letter’ folder. </p>
<p>Then I start writing.</p>
<p>Sometimes I don’t have time to write, so I’ll just create a document with a headline. Then, if I still care about the idea later, I can write it then.</p>
<p>When something is published. I right click on the text document in the column on the left, and mark as ‘published’.</p>
<p>I have another scrivener document on my desktop called ‘Untether’. This is where I’m booking my next book. It is dedicated exclusively to Untether (more about the the <a href="/posts/untethering">Untether project</a>). If I write something for Untether by accident in my Notebook, I only need to copy and paste it over. I keep these separate, because Untether is a separate project from my notebook.</p>
