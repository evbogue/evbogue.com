# Agents

Role files for specialized agents working on evbogue.com.

Use these when you want an agent to take on a specific job instead of the whole project:

- `WRITER.md` — turn rough notes or drafts into publishable blog posts.
- `EDITOR.md` — sharpen drafts into Everett's punchier 2010/Gawker-informed voice.
- `DESIGNER.md` — keep the site readable, minimal, and publication-first.
- `CODER.md` — implement small blog features and tooling without bloating the stack.
- `DEVOPS.md` — keep the VPS, pull process, and live routes healthy.
- `SOCIAL-MEDIA-MANAGER.md` — turn finished posts into sharp distribution copy.
- `ACCOUNT-MANAGER.md` — handle subscribers, replies, sponsors, and reader relationships.
- `PRODUCT-MANAGER.md` — prioritize the publishing system roadmap.
- `ARCHIVIST.md` — classify, restore, contextualize, and rewrite the full blog archive.
- `RESTORATIONIST.md` — clean recovered archive essays in small batches without rewriting history.
- `ANALYST.md` — first-party analytics, weekly report, signal over vanity.

Keep role files practical. Each one should explain the job, the standard of quality, what files it should touch, and what to report back.
