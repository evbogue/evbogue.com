The blog has a newsletter again. The signup form is at the bottom of every page. The whole subscriber list is a JSON file on my server.

The flow is what you would want. Type your email, get a confirmation link, click it, you are in. Every email I send has a one-click unsubscribe at the bottom. The token is the auth. The file is written by renaming a temp file over the live one, which is atomic, which means the list can never end up half-written.

I did not type the code. Claude wrote most of it. I told it what should exist, it produced a version, I read what it produced, I asked for changes. The judgment was mine. The typing was the agent's.

This was not possible at this difficulty level before. Running your own mailing list used to be a sysadmin job, then a weekend of engineering work, then a recurring fee because nobody wanted the weekend. The cost stayed high enough that doing it yourself was the strange choice.

Agentic harnesses are the first tool that drops the cost to one afternoon of conversation. Not faster typing. Not better libraries. A new kind of leverage. The first lever where a writer can also run the infrastructure that delivers what they wrote.

The newsletter is real. The next dispatch goes out from Chicago.
