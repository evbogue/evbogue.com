This is the first post on the new evbogue.com.

The plan is simple: write in Obsidian, drop finished markdown files into `posts/`, push to GitHub, let the VPS pull and serve. No build step, no CMS, no lock-in — just files on disk and a tiny Deno server rendering them on request.
